Data Source

The data used in this app is sourced from a .csv file available at SimpleMaps (basic and free version).

Distance Calculation

Distances are calculated using the Haversine formula.

How It Works

If you select a country without choosing a specific city, the app determines the city within that country with the highest number of capitals closer to it than to its own capital. This involves iterating over all cities in the country and calculating the distance to all capitals.

If you select a specific city within a country, the app calculates and displays the distances from that chosen city to all the world's capitals that are closer to this city than its own capital is.

This process can be time-consuming if you are the first person to request data for a specific country. Once a 'winning city' for a country is determined, the result is stored in a database, leading to faster access for subsequent requests.

Stored Countries

On the "Stored Countries" page, you can view all countries for which a request has already been processed, along with the results.
